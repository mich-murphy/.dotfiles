## Installing
Requirements:
- `git`
- `stow`

Clone repository
```
git clone https://github.com/mrhackendbacker/Hackfiles.git ~
```

Run `stow` to symlink all folders (ignoring README.md etc.)
```
stow */
```
Or `stow` individual folders e.g.
```
stow nvim
```
